## Boolean Condition

### Description:
Evaluates boolean expressions and routes the workflow based on the evaluation result.

Key Parameters:

> **data.action** - Typically set to `"condition"` to indicate the evaluation action.

> **params.condition** - Array of condition objects that define the logic to evaluate.

> **params.trueNode** - Identifier for the node to execute if the condition evaluates to true.

> **params.falseNode** - Identifier for the node to execute if the condition evaluates to false.

Possible unit actions:

1. **equal** - Compares two values for equality.
2. **more** - Compares two values to check if the left value is greater than the right value.
3. **less** - Compares two values to check if the left value is less than the right value.
4. **more or equal** - Compares two values to check if the left value is greater than or equal to the right value.
5. **less or equal** - Compares two values to check if the left value is less than or equal to the right value.
6. **not equal** - Compares two values to check if they are not equal.

Connections:

- **Input:** Receives data to be evaluated.
- **Outputs:** Provides two branches â€” one for a true outcome and another for a false outcome.

Use Cases:

- Implementing decision logic within workflows, e.g., routing data based on status or other criteria.

### Code Example

1. **block_id** - Unique identifier for the Boolean Condition block.
2. **connected_true_block_id** - Unique identifier for the block to follow if the condition is true.
3. **connected_false_block_id** - Unique identifier for the block to follow if the condition is false.
4. **condition_id** - Unique identifier for the condition.
5. **unit_id** - Unique identifier for the unit.

```json
{
  "_id": "block_id",
  "version": "1.0",
  "type": "BOOLEAN_CONDITION",
  "payload": {
    "name": "boolean-condition-0",
    "data": {
      "action": "condition",
      "params": {
        "params": {
          "trueNode": "connected_true_block_id",
          "falseNode": "connected_false_block_id",
          "condition": [
            {
              "id": "condition_id",
              "unit": [
                {
                  "id": "unit_id",
                  "left": {
                    "value": "1",
                    "type": "string"
                  },
                  "right": {
                    "value": "1",
                    "type": "string"
                  },
                  "action": "equal"
                }
              ]
            }
          ]
        }
      }
    }
  },
  "connection": {
    "nodes": [
      "connected_true_block_id",
      "connected_false_block_id"
    ]
  }
}
