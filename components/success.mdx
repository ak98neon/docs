<Info>This block marks the successful completion of a workflow.</Info>

## Success

### Description:
Indicates that the pipeline execution has been completed successfully.

Key Parameters:

- No additional parameters required.

Connections:

- **Input:** Receives execution from preceding blocks.
- **No Output:** This is a terminal block.

Use Cases:

- Signaling that a process has finished correctly.
- Ending a workflow when all operations complete successfully.
